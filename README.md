# Telecom Customer Churn Analysis

## Project Overview
This project analyzes customer churn behavior in a telecom company using Python and Exploratory Data Analysis (EDA). The goal is to identify the key factors contributing to customer attrition and provide actionable business insights to improve customer retention.

## Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Key Findings
- Overall churn rate is approximately **26.5%**.
- Customers on **Month-to-Month contracts** are more likely to churn.
- **Fiber Optic** users show higher churn compared to other internet service types.
- Customers without **Online Security, Tech Support, Device Protection, and Online Backup** have significantly higher churn rates.
- Customers with shorter tenure are more likely to leave the service.

## Business Recommendations
- Promote long-term contracts through incentives.
- Encourage adoption of value-added services.
- Improve onboarding and engagement for new customers.
- Target high-risk customer segments with retention campaigns.
